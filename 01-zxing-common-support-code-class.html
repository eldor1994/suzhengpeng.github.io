Package com.google.zxing

Class Description
Binarizer This class hierarchy provides a set of methods to convert luminance data to 1 bit data.
提供一系列的方法将灰度值二值化处理
BinaryBitmap This class is the core bitmap class used by ZXing to represent 1 bit data.
Dimension Simply encapsulates a width and height.
简单的封装宽度和高度
HtmlAssetTranslator A utility which auto-translates the English-language text in a directory of HTML documents using Google Translate.
InvertedLuminanceSource A wrapper implementation of LuminanceSource which inverts the luminances it returns – black becomes white and vice versa, and each value becomes (255-value).
将灰度值进行翻转。白->黑 黑->白
LuminanceSource The purpose of this class hierarchy is to abstract different bitmap implementations across platforms into a standard interface for requesting greyscale luminance values.
该类层次结构的目的是将跨平台的不同位图实现抽象为用于请求灰度亮度值的标准接口。
MultiFormatReader MultiFormatReader is a convenience class and the main entry point into the library for most uses.
MultiFormatWriter This is a factory class which finds the appropriate Writer subclass for the BarcodeFormat requested and encodes the barcode with the supplied contents.
PlanarYUVLuminanceSource This object extends LuminanceSource around an array of YUV data returned from the camera driver, with the option to crop to a rectangle within the full data.
该对象将LuminanceSource延伸到从相机驱动程序返回的YUV数据数组中，并可以选择在完整数据内裁剪到矩形。
Result Encapsulates the result of decoding a barcode within an image.
ResultPoint Encapsulates a point of interest in an image containing a barcode.
RGBLuminanceSource This class is used to help decode images from files which arrive as RGB data from an ARGB pixel array.
该类用于解码从argb像素阵列到RGB数据的图像文件
StringsResourceTranslator A utility which auto-translates English strings in Android string resources using Google Translate.
